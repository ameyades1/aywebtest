# Claude Code Prompt: Generate v7 and v8 with White Copper Colors

## Task

Create two new HTML files — `frontend-branded-antaryog-v7.html` and `frontend-branded-antaryog-v8.html` — in `docs/prompts/output/`.

These are **not restyled versions** of the old v7/v8. They are **brand new files** built by taking the complete HTML and CSS structure of the existing files listed below, keeping all fonts exactly as they are, and replacing only the color scheme with the White Copper palette defined in this prompt.

- Base `v7` on: `docs/prompts/output/frontend-branded-antaryog-v5.html` *(or any odd-numbered minimal version — single-column, core layout)*
- Base `v8` on: `docs/prompts/output/frontend-branded-antaryog-v6.html` *(or any even-numbered full-content version — expanded layout with more program details)*

---

## What Must NOT Change

| Category | Rule |
|---|---|
| **Fonts** | Keep every `font-family` declaration and every Google Fonts `<link>` tag exactly as found in the source file. Do not add or remove any typeface. |
| **Font sizing** | No changes to `font-size`, `line-height`, `letter-spacing`, or `font-weight`. |
| **HTML content** | Every word stays as written — headings, body copy, nav labels, button text, stat numbers, testimonials, program names, footer links. |
| **HTML structure** | No elements added, moved, or removed (other than the logo swap above). |
| **Class names & IDs** | Unchanged. |
| **Layout & spacing** | No changes to `grid`, `flex`, `padding`, `margin`, `width`, `height`, or `gap`. |
| **Animations** | Preserve all `@keyframes` and `transition` values. Only update color values used within them. |
| **JavaScript** | No changes. |
