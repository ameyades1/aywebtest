## Asset Optimization Guidelines

### Image Compression Standards

| Asset Type | Format | Max File Size | Quality | Tool |
|---|---|---|---|---|
| Logo | PNG | 50KB | Lossless | TinyPNG, ImageOptim |
| Portrait Photos | JPEG | 150KB | 85% | JPEGmini, Squoosh |
| Program Cards | JPEG/WebP | 100KB | 80% | Squoosh, WebP converter |
| Event Cards | JPEG/WebP | 100KB | 80% | Squoosh, WebP converter |
| Section Images | JPEG/WebP | 200KB | 85% | Squoosh, WebP converter |

### Responsive Image Strategy

**Implement srcset for responsive delivery:**

```html
<img
  src="/assets/programs/discourses-vedanta.jpg"
  srcset="/assets/programs/discourses-vedanta-400w.jpg 400w,
          /assets/programs/discourses-vedanta-800w.jpg 800w,
          /assets/programs/discourses-vedanta-1200w.jpg 1200w"
  sizes="(max-width: 768px) 100vw,
         (max-width: 1024px) 50vw,
         33vw"
  alt="Vedanta Discourses by Acharya Upendra Ji"
  loading="lazy"
>
```

**Generate Multiple Sizes:**
- Small: 400px width (mobile)
- Medium: 800px width (tablet)
- Large: 1200px width (desktop)

### WebP Conversion

**Why WebP?**
- 25-35% smaller file size than JPEG
- Maintains quality
- Supported by 95%+ modern browsers

**Conversion Tools:**
- Squoosh.app (online)
- `cwebp` command-line tool
- Photoshop/GIMP WebP plugins

**Fallback Strategy:**
```html
<picture>
  <source srcset="image.webp" type="image/webp">
  <img src="image.jpg" alt="Description">
</picture>
```

1. **Unsplash** (unsplash.com)
   - Search: "spiritual india", "vedanta", "yoga india"
   - High quality, free for commercial use

2. **Pexels** (pexels.com)
   - Similar to Unsplash
   - Good Indian spiritual content
   - Free for commercial use

3. **Pixabay** (pixabay.com)
   - Free images and videos
   - Good for Indian cultural content
   - No attribution required

**Premium Stock (If Budget Available):**
1. **Shutterstock**
   - Extensive Indian spiritual collection
   - High quality, curated content
   - License required (~$29-$99/image)

2. **Adobe Stock**
   - Professional quality

   - Subscription or per-image pricing
